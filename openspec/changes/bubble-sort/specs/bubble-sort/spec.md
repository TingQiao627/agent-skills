# Spec: 冒泡排序

## 功能描述
实现冒泡排序算法，对整数数组进行升序排序。

## 需求

### REQ-001: 基本排序功能
- **描述**: 函数接受一个整数数组，返回升序排列后的数组
- **输入**: `number[]` - 整数数组
- **输出**: `number[]` - 升序排列的整数数组
- **验收标准**:
  - 输入 `[5, 3, 8, 1, 2]` → 输出 `[1, 2, 3, 5, 8]`
  - 输入 `[1, 2, 3, 4, 5]` → 输出 `[1, 2, 3, 4, 5]`（已排序数组）

### REQ-002: 空数组处理
- **描述**: 空数组输入应返回空数组
- **输入**: `[]`
- **输出**: `[]`

### REQ-003: 单元素数组处理
- **描述**: 单元素数组应返回自身
- **输入**: `[1]`
- **输出**: `[1]`

### REQ-004: 重复元素处理
- **描述**: 包含重复元素的数组应正确排序
- **输入**: `[3, 3, 3, 1, 2]`
- **输出**: `[1, 2, 3, 3, 3]`

### REQ-005: 负数处理
- **描述**: 包含负数的数组应正确排序
- **输入**: `[-1, -3, 0, 5, 2]`
- **输出**: `[-3, -1, 0, 2, 5]`

### REQ-006: 逆序数组处理
- **描述**: 完全逆序的数组应正确排序
- **输入**: `[5, 4, 3, 2, 1]`
- **输出**: `[1, 2, 3, 4, 5]`

## 非功能性需求
- 时间复杂度: O(n²)
- 空间复杂度: O(1)（原地排序）
- 稳定性: 稳定排序（相等元素保持原有相对顺序）
